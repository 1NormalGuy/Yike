# Jiang Yike's Secret

一个完全离线、面向平板横屏触控的像素风礼物互动小游戏。

## 后台运行（推荐）

在项目目录执行：

```bash
./scripts/start-server.sh
```

脚本会明确监听 `0.0.0.0:8080`，并打印本机地址和局域网地址，例如：

```text
Local:   http://127.0.0.1:8086
LAN:     http://192.168.1.13:8086
```

停止后台服务：

```bash
./scripts/stop-server.sh
```

也可以指定其他端口：

```bash
./scripts/start-server.sh 8090
./scripts/stop-server.sh 8090
```

平板和电脑需要连接同一个局域网。如果 macOS 弹出防火墙提示，需要允许 Python 接受传入连接。

## 移动端使用

- 横屏触控设备会自动铺满当前浏览器可视区域，不再固定为桌面端 `16:9` 画布。
- 首页和房间右上方提供“全屏”入口。Android 等支持 Fullscreen API 的浏览器可直接进入全屏。
- iPhone / iPad 浏览器不允许普通网页按钮直接隐藏地址栏时，点击分享并选择“添加到主屏幕”，再从桌面图标打开即可全屏运行。
- 首页使用 `music/小情歌.mp3`。浏览器阻止有声自动播放时，点击首页的“播放音乐”即可开始。

## 游戏说明

- 五个小游戏：星光接力、愤怒小狗出击（弹射小狗击退黑猫）、治愈连连看、欧气盲盒机、AI 心动便利贴。
- 完成全部五个小游戏后解锁中央召唤阵。
- 最终召唤密码是 `0612`（不再显示在召唤阵中央，需要自己记住这个特别的日子）。
- 游戏进度保存在浏览器 `localStorage` 中。
- 页面右下角的“重置进度”可清空全部游戏状态；操作前会二次确认。
- 连续点击开始页标题或页面右上角隐藏区域 5 次仍可作为备用重置方式。
- 字体（Press Start 2P / Zpix 像素字）通过 CDN 加载；离线时自动回退到本地等宽字体，游戏逻辑不依赖任何接口。
